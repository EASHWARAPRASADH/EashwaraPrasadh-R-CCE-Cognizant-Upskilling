# Core Java: Arrays and Collections Module

This module demonstrates data structures in Java, ranging from simple static arrays to dynamic Collections frameworks such as `ArrayList` and `HashMap`.

## Exercises Checklist
1. `ArraySumAverage.java`: Demonstrates array initialization, iteration, sum calculation, and mathematical average calculation for double values.
2. `ArrayListExample.java`: Illustrates dynamic arrays using `ArrayList`, including adding elements, searching, sorting via `Collections.sort`, and printing items.
3. `HashMapExample.java`: Showcases key-value storage using `HashMap`, allowing mapping from unique student IDs to student names and searching entries dynamically.

## Compilation and Execution
To compile and execute any of these files, navigate to this directory and run:
```bash
javac FileName.java
java FileName
```
For example, to run the HashMap Example:
```bash
javac HashMapExample.java
java HashMapExample
```
