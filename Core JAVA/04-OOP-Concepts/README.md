# Core Java: OOP Concepts Module

This module demonstrates Object-Oriented Programming (OOP) concepts in Java, including encapsulation, inheritance, interfaces, polymorphism (method overloading), and class constructors.

## Exercises Checklist
1. `CarDemo.java`: Showcases encapsulation, private variables, public getters/setters, constructors, and object instantiation for a simple `Car` class.
2. `InheritanceDemo.java`: Illustrates inheritance using a parent class `Animal` and a child class `Dog` that overrides the `sound()` method.
3. `InterfaceDemo.java`: Demonstrates loose coupling and polymorphism using an interface `Vehicle` and its implementations (`Car` and `Bike`).
4. `MethodOverloading.java`: Shows compile-time polymorphism through method overloading, defining multiple `add` methods with different parameters and types.

## Compilation and Execution
To compile and execute any of these files, navigate to this directory and run:
```bash
javac FileName.java
java FileName
```
For example, to run the Interface Demo:
```bash
javac InterfaceDemo.java
java InterfaceDemo
```
