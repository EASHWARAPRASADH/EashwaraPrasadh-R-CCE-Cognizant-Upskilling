# Core Java: Control Flow Module

This module demonstrates logical branching, looping constructs, user inputs, and basic algorithmic validation in Java.

## Exercises Checklist
1. `EvenOdd.java`: Simple logical check using modulo operator to determine if an integer is even or odd.
2. `GradeCalculator.java`: Uses nested `if-else` branches to output academic grades (A, B, C, D, F) based on user-provided score marks.
3. `GuessGame.java`: An interactive guessing game using a `while` loop that requests input until the user guesses a predefined secret number.
4. `LeapYear.java`: Standard leap year check using logical AND (`&&`) and OR (`||`) operators to check alignment with the Gregorian calendar rules.
5. `MultiplicationTable.java`: Outputs a multiplication grid (up to 10) for a given number using a standard `for` loop.

## Compilation and Execution
To compile and execute any of these files, navigate to this directory and run:
```bash
javac FileName.java
java FileName
```
For example, to run the Guess Game:
```bash
javac GuessGame.java
java GuessGame
```
