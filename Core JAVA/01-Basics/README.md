# Core Java: Basics Module

This module covers the absolute basics of Java programming, including variables, data types, operators, control flow statements, recursion, and string manipulation.

## Exercises Checklist
1. `HelloWorld.java`: The traditional entry-point program showing how to print text to the standard console output.
2. `DataTypesDemo.java`: Demonstrates the usage and range of primitive data types in Java (byte, short, int, long, float, double, char, boolean) along with printing variable values.
3. `OperatorPrecedence.java`: Explains the hierarchy and precedence of operators (such as arithmetic, unary, and logical operators) in mathematical calculations.
4. `TypeCasting.java`: Demonstrates implicit (widening) type casting and explicit (narrowing) type casting in Java.
5. `SimpleCalculator.java`: Interactive command-line calculator utilizing standard arithmetic operators and a `switch` case construct.
6. `StringReverse.java`: Demonstrates standard and programmatic techniques to reverse a string value.
7. `PalindromeChecker.java`: Validates if a user-input word is a palindrome (reads the same backward as forward).
8. `FactorialCalculator.java`: Calculates the factorial of a positive integer using iteration (loops).
9. `RecursiveFibonacci.java`: Calculates Fibonacci sequence numbers using recursion.
10. `MoneyTransfer.java`: A simple simulator tracking account deposits, balance adjustments, and transfer logic.
11. `Demo.java`: A basic test class showcasing output methods.

## Compilation and Execution
To compile and execute any of these files, navigate to this directory and run:
```bash
javac FileName.java
java FileName
```
For example, to run the Palindrome Checker:
```bash
javac PalindromeChecker.java
java PalindromeChecker
```
