# Core Java: Advanced Concepts Module

This module demonstrates advanced language features introduced in modern releases of Java, including functional programming (Lambdas & Streams), Records, Pattern Matching for Switch, and runtime class inspection via Reflection.

## Exercises Checklist
1. `LambdaExample.java`: Illustrates functional interfaces and Lambda expressions used for concise implementation of runnable interfaces or math operations.
2. `StreamExample.java`: Demonstrates stream pipelines for filtering, mapping, and collecting elements from collections in a functional manner.
3. `RecordExample.java`: Showcases how to define immutable data carrier classes using Java's `record` keyword, including auto-generated constructors, equals, hashCode, and toString methods.
4. `PatternMatchingSwitch.java`: Shows the usage of pattern matching inside `switch` statements to test object types and automatically bind variables.
5. `ReflectionExample.java`: Illustrates the Java Reflection API, allowing inspection of constructors, fields, and methods of a class at runtime.

## Compilation and Execution
To compile and execute any of these files, navigate to this directory and run:
```bash
javac FileName.java
java FileName
```
For example, to run the Stream Example:
```bash
javac StreamExample.java
java StreamExample
```
