# Core Java: File I/O Module

This module demonstrates file persistence and I/O streams in Java. It covers writing and reading text contents to and from the local filesystem.

## Exercises Checklist
1. `FileWriting.java`: Shows how to create and write dynamic text content to a local file (`output.txt`) using the standard `FileWriter` and `BufferedWriter` wrappers.
2. `FileReading.java`: Illustrates opening, buffering, and reading text files line-by-line using `FileReader` and `BufferedReader`.

## Compilation and Execution
To compile and execute any of these files, navigate to this directory and run:
```bash
javac FileName.java
java FileName
```
For example, to run File Writing first, and then File Reading:
```bash
javac FileWriting.java
java FileWriting

javac FileReading.java
java FileReading
```
