This comment demonstrates aligning images to the left of block content. Since Bootstrap 5 dropped the legacy .media class, we implement this using standard flex alignment classes like d-flex and flexbox growth settings.

I learned how to implement media alignment grids (comment lists/avatars) using Bootstrap 5 flexbox classes like flex-shrink-0 and flex-grow-1.
