Exploring Bootstrap Directory Structure

Understanding what is inside the downloaded package.

CSS Folder

Contains style files. bootstrap.css contains the main styles. bootstrap.min.css is the minified (compressed) version which loads faster in production. .map files are used for debugging.

JS Folder

Contains JavaScript files for interactive components like dropdowns, modals, and tooltips. bootstrap.bundle.js includes Popper.js, which is required for positioning dropdowns and popovers.

What I learned from this exercise:

I explored the contents of the compiled Bootstrap package and identified the CSS and JS directories. I learned the difference between regular and minified (.min) files, and why bundles are preferred (since they package dependencies like Popper.js).
