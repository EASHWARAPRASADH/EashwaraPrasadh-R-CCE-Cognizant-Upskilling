## How to View the Exercises
Open the `index.html` file inside any of the exercise sub-folders in your web browser. 

*Note: For Sass customization exercises, run `npm install` and `npm run build-css` inside the respective directories (`Ex19.1-Sass-Setup` or `Ex19.2-Custom-Sass`) to recompile the custom theme styles.*
